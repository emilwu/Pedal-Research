# 吉他設備評比專案 - 標準化評估框架

**版本:** 1.0  
**最後更新:** 2025-10-28  
**適用範圍:** 吉他踏板、音箱、綜合效果器等設備評比

---

## 核心評比原則

### 1. 使用者導向原則
所有評比必須基於使用者的實際需求,包含:
- **現有設備配置** - 使用者已擁有的吉他、音箱、效果器
- **演奏風格** - Jazz、Neo Soul、Rock、Post Rock、Fusion、Funk、Pop Rock 等
- **使用情境** - 居家練習、錄音、現場演出
- **預算限制** - 明確的價格範圍

### 2. 互補性原則
- **最大互補、最小重疊** - 新設備應填補現有配置的音色空白
- **功能區隔** - 避免與現有設備產生功能冗餘
- **音色三角** - 建構完整的音色覆蓋範圍

### 3. 客觀性原則
- **搜尋驗證** - 必須使用 web_search 搜尋產品規格、評價、比較
- **多元來源** - 參考專業評測、用戶評價、官方規格
- **引用來源** - 提供可追溯的參考連結

## 評比框架結構

### 階段一：資訊收集

#### 1.1 使用者背景分析
收集以下資訊:
```
□ 現有吉他清單(型號、拾音器配置、琴身木材、指板材質)
□ 現有音箱/效果器配置
□ 主要演奏風格(依重要性排序)
□ 使用情境(居家/錄音/演出,比重分配)
□ 預算範圍
□ 特殊需求或限制
```

#### 1.2 產品資訊搜尋
必須搜尋的資訊類別:
```
□ 官方規格(技術參數、控制選項)
□ 音色特性(頻率響應、動態範圍、增益結構)
□ 專業評測(音色描述、使用情境、優缺點)
□ 用戶評價(長期使用經驗、常見問題)
□ 價格資訊(建議售價、市場行情、二手價格)
□ 版本差異(如有多個版本,需比較差異)
```

### 階段二:規格分析

#### 2.1 技術規格評估
| 評估項目 | 音箱 | 踏板 | 綜合效果器 |
|---------|------|------|-----------|
| **核心技術** | 管機/晶體/建模 | 類比/數位/混合 | 處理器類型/DSP規格 |
| **功率/訊號** | 瓦數/阻抗 | 輸入/輸出阻抗 | 訊號路徑/延遲 |
| **音色塑造** | 前級/EQ/效果 | 控制旋鈕/開關 | 效果類型/可調參數 |
| **連接性** | 輸入/輸出/Loop | 輸入/輸出/MIDI | I/O配置/擴充性 |
| **便利性** | 重量/尺寸 | 尺寸/供電 | 編輯介面/預設管理 |

#### 2.2 音色特性分析
使用統一描述框架:
```
頻率特性:
├─ 低頻: [薄弱/適中/飽滿/過度] + 描述
├─ 中頻: [削減/中性/突出/激進] + 描述  
└─ 高頻: [黯淡/平滑/明亮/刺耳] + 描述

動態響應:
├─ 撥弦敏感度: [低/中/高]
├─ 音量敏感度: [低/中/高]
└─ Compression: [無/輕微/適中/強烈]

失真特性(如適用):
├─ Breakup 點: [低/中/高] + 音量值
├─ 增益範圍: [輕微/中等/高/極高]
└─ 諧波特性: [偶次/奇次/豐富/乾淨]
```

### 階段三:適配性評估

#### 3.1 與現有設備的搭配評估
針對每款現有吉他分析:
```
吉他型號: [名稱]
├─ 拾音器特性: [輸出/頻率/動態]
├─ 搭配評分: ★★★★★ (1-5星)
├─ 音色互補性: [描述]
├─ 預期音色結果: [描述]
└─ 注意事項: [特殊考量]
```

#### 3.2 音樂風格適配性
針對每種演奏風格評估:
```
風格: [Jazz/Rock/Fusion等]
├─ 適配度: ★★★★★ (1-5星)
├─ 理由: [為何適合/不適合]
├─ 音色需求滿足: [詳述]
└─ 替代選擇: [如不理想,建議其他選項]
```

#### 3.3 使用情境適配性
```
居家練習: [適合度★★★★★ + 原因]
錄音應用: [適合度★★★★★ + 原因]
現場演出: [適合度★★★★★ + 原因]
```

### 階段四:比較評分

#### 4.1 綜合評分系統
使用100分制,各項目權重可依需求調整:

**音色品質 (30分)**
- 音色特性 (15分) - 頻率平衡、諧波豐富度
- 動態響應 (15分) - 撥弦敏感度、音量敏感度

**功能性 (25分)**
- 音色可調性 (10分) - 參數範圍、控制精細度
- 實用性 (10分) - 操作簡便性、切換速度
- 擴充性 (5分) - MIDI/連接選項

**互補性 (25分)**
- 與現有設備互補 (15分) - 填補音色空白
- 風格匹配度 (10分) - 符合演奏需求

**性價比 (15分)**
- 價格合理性 (10分) - 相對市場定位
- 投資價值 (5分) - 保值性、升級空間

**便利性 (5分)**
- 尺寸重量 (2分)
- 供電/維護 (3分)

#### 4.2 排名原則
- 依綜合評分排序
- 同分時優先考慮「互補性」分數
- 明確標示「推薦」、「可考慮」、「不推薦」

### 階段五:報告產出

#### 5.1 報告結構
```
1. 執行摘要
   ├─ 首選推薦(1-3款,含分數)
   ├─ 次要選擇(依需求)
   └─ 不推薦項目(含原因)

2. 詳細評比
   ├─ 每款設備的完整分析
   │   ├─ 核心規格
   │   ├─ 音色特性
   │   ├─ 搭配評估
   │   ├─ 適配性分析
   │   └─ 評分明細
   └─ 橫向比較表

3. 購買建議
   ├─ 最佳選擇情境
   ├─ 預算考量建議
   ├─ 組合購買建議(如適用)
   └─ 注意事項

4. 參考資料
   └─ 所有參考連結(官網、評測、用戶評價)
```

#### 5.2 寫作風格
- **簡潔專業** - 避免冗長描述,直接提供關鍵資訊
- **數據支持** - 用具體數字和評分代替模糊描述
- **平衡呈現** - 同時說明優點與缺點
- **實用導向** - 提供可行動的建議,而非僅理論分析

## 特殊設備類型的評比重點

### 音箱評比額外重點
```
□ Clean headroom - 對Jazz/Neo Soul尤其重要
□ 喇叭配置 - 尺寸、品牌、音色影響
□ 功率段切換 - 居家使用的必要性
□ 內建效果 - Reverb/Tremolo品質
□ 錄音輸出 - DI/Line out/IR選項
□ 重量與尺寸 - 實際可攜性考量
```

### 踏板評比額外重點
```
□ True Bypass vs Buffered
□ 供電需求 - 電流/電壓/極性
□ 疊加兼容性 - 與其他踏板的互動
□ 雜訊底噪 - 訊噪比測試
□ 旁路音色 - 對整體音色的影響
□ 尺寸與Pedalboard整合性
```

### 綜合效果器評比額外重點
```
□ DSP運算能力 - 同時效果數量
□ 延遲表現 - 訊號延遲毫秒數
□ 預設管理 - 儲存/切換/編輯便利性
□ 音箱模擬品質 - IR載入/cabinet sim
□ MIDI整合 - 控制選項/擴充性
□ 韌體更新 - 廠商支援/社群活躍度
```

## 品質控制檢查清單

評比報告產出前,必須檢查:

### 內容完整性
```
□ 所有產品都有搜尋驗證
□ 每款產品都有完整評分
□ 適配性分析涵蓋所有使用者吉他
□ 音樂風格適配性完整評估
□ 參考連結全部有效
```

### 邏輯一致性
```
□ 評分標準統一應用
□ 推薦理由與評分一致
□ 互補性分析符合「最大互補、最小重疊」原則
□ 價格與評分合理對應
□ 不推薦理由明確且充分
```

### 實用性
```
□ 提供明確的「最佳選擇」
□ 包含預算考量建議
□ 說明購買優先順序
□ 提供備選方案
□ 警示潛在問題
```

## 特殊情境處理

### 當搜尋結果不足時
1. 擴大搜尋範圍至相似產品
2. 搜尋品牌系列的整體評價
3. 尋找替代資訊來源(如廠商白皮書)
4. 明確標註「資訊有限」並降低推薦信心度

### 當產品已停產時
1. 註明停產狀態
2. 搜尋二手市場價格
3. 提供現行替代產品
4. 評估長期支援與零件可得性

### 當出現利益衝突資訊時
1. 優先採用多數共識
2. 標註爭議性意見
3. 提供多元觀點
4. 讓使用者自行判斷

### 當預算嚴重限制選項時
1. 誠實說明理想選擇
2. 提供預算內最佳方案
3. 說明妥協之處
4. 建議分階段購買計畫

## 附錄:標準術語定義

### 音色描述術語
- **Warm** (溫暖) - 低中頻豐富,高頻柔和
- **Bright** (明亮) - 高頻突出,清晰度高
- **Dark** (黯淡) - 高頻削減,低頻為主
- **Thick** (厚實) - 低頻與低中頻飽滿
- **Thin** (單薄) - 低頻不足,中高頻為主
- **Transparent** (透明) - 保留原始音色特性
- **Colored** (染色) - 明顯改變音色特性

### 動態術語
- **Touch-sensitive** (觸鍵敏感) - 對撥弦力度反應靈敏
- **Compressed** (壓縮) - 動態範圍縮小
- **Dynamic range** (動態範圍) - 最輕到最重撥弦的音量差
- **Headroom** (餘裕空間) - 不失真的最大音量能力

### 技術術語
- **Clean tone** (清音) - 未失真的原始音色
- **Breakup** (破音點) - 開始產生失真的音量點
- **Gain staging** (增益級聯) - 訊號鏈中的增益分配
- **Signal chain** (訊號鏈) - 吉他→效果器→音箱的訊號路徑
