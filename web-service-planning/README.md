# Pedal Research 網頁服務規劃

本目錄包含將 Pedal Research AI Agent 系統轉化為公開網頁服務的完整規劃文件。

## 文件結構

| 文件 | 說明 |
|-----|------|
| [01-product-vision.md](./01-product-vision.md) | 產品願景、核心價值、目標用戶 |
| [02-mvp-scope.md](./02-mvp-scope.md) | MVP 版本範圍定義與功能清單 |
| [03-modules-specification.md](./03-modules-specification.md) | 各模組詳細規格說明 |
| [04-user-flows.md](./04-user-flows.md) | 核心用戶流程與互動設計 |
| [05-future-roadmap.md](./05-future-roadmap.md) | 後續版本路線圖與商業模式 |
| [appendix/existing-system-summary.md](./appendix/existing-system-summary.md) | 現有 AI Agent 系統摘要（供開發參考） |

## 閱讀順序建議

1. 先閱讀 **01-product-vision.md** 了解整體方向
2. 閱讀 **02-mvp-scope.md** 確認 MVP 範圍
3. 閱讀 **03-modules-specification.md** 了解各模組細節
4. 閱讀 **04-user-flows.md** 理解用戶互動邏輯
5. 閱讀 **05-future-roadmap.md** 了解後續擴展方向
6. 需要時參考 **appendix/** 中的現有系統資料

## 重要說明

- 本規劃採用 **MVP (Minimum Viable Product)** 思維
- 所有文件專注於**功能規格與流程設計**
- **不包含技術實作細節**（技術棧選擇留給開發階段決定）
- 設計保留後續擴展彈性
