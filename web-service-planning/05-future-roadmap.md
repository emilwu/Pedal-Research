## v1.5：商業化基礎

### 新增功能

#### 1. 訂閱系統
```
用戶類型
├── 免費用戶
│   ├── 設備數據庫：公開（唯讀）
│   ├── 設備研究：共享額度排隊
│   └── 智慧配對：每日有限次數
│
└── 訂閱用戶
    ├── 設備數據庫：可私人化
    ├── 設備研究：每月限額直接處理
    ├── 智慧配對：每日較高限額
    └── 訊號鏈生成：每日限額（v2.0 新增）
```

#### 2. 付費整合
- 訂閱方案管理
- 付款處理
- 發票/收據
- 訂閱續費/取消

#### 3. 設備研究排隊優化
```
免費用戶請求流程：
提交請求 → 加入共享隊列 → 顯示等待時間 → [可選] 單次付費插隊
                                              │
                                              ▼
                                         立即開始處理

訂閱用戶請求流程：
提交請求 → 檢查月額度 → 有額度 → 直接開始處理
                      │
                      ▼
                    額度用完 → 加入共享隊列
```

#### 4. 私人設備數據庫
- 訂閱用戶可建立私人數據庫
- 私人研究結果不公開
- 可選擇將私人設備公開（需審核）

#### 5. 內容審核機制
- 用戶公開設備需審核
- 審核隊列管理
- 審核標準制定

### 商業模式設計

#### 訂閱方案（參考）

| 項目 | 免費 | 基礎訂閱 | 進階訂閱 |
|-----|-----|---------|---------|
| 月費 | $0 | $X | $Y |
| 數據庫 | 公開 | 可私人化 | 可私人化 |
| 設備研究 | 排隊 | N 次/月直接 | M 次/月直接 |
| 智慧配對 | A 次/日 | B 次/日 | C 次/日 |
| 訊號鏈生成 | - | D 次/日 | E 次/日 |

> 注：具體數字待根據成本和市場驗證後決定。

#### 單次付費選項
- 設備研究插隊：$X/次
- 額外配對次數：$X/N次

## v2.0：進階功能

### 新增功能

#### 1. 訊號鏈生成（P4 功能）

**功能說明**
根據用戶的完整設備清單和音樂風格，生成完整的訊號鏈配置。

**輸出內容**
- 完整訊號流程圖（視覺化）
- 各效果器連接順序
- 各效果器詳細設定
- 4CM / Pre-Amp 方法建議
- 訊號路徑說明

**依賴**
- 完整的設備數據庫資料
- 智慧配對結果
- pairing_rules 知識庫

#### 2. 設備替代性分析

**功能說明**
分析用戶設備中的功能重疊，並建議替代方案。

**輸出內容**
- 功能重疊報告
- 替代設備建議
- 精簡方案推薦
- 升級路徑建議

#### 3. 配對結果比較

**功能說明**
允許用戶比較不同配對結果的差異。

**使用情境**
- 比較不同吉他對同一風格的配對差異
- 比較不同風格對同一設備組合的差異

### 限額設計

| 功能 | 免費 | 基礎訂閱 | 進階訂閱 |
|-----|-----|---------|---------|
| 訊號鏈生成 | - | N 次/日 | M 次/日 |
| 替代分析 | - | N 次/月 | M 次/月 |
| 結果比較 | 無限 | 無限 | 無限 |

## v3.0：社群擴展

### 新增功能

#### 1. 社群功能

**配置分享**
- 用戶可公開分享配對結果
- 分享連結生成
- 社群瀏覽頁面

**評價系統**
- 對設備評分（1-5星）
- 用戶評論
- 熱門設備排行

**用戶互動**
- 追蹤其他用戶
- 收藏其他用戶的配置
- 討論區（簡易）

#### 2. 多語言支援

**第一階段**
- 英文介面

**後續階段**
- 日文
- 其他語言（依需求）

#### 3. 進階數據分析

**用戶洞察**
- 最受歡迎的設備組合
- 風格趨勢分析
- 個人化推薦

**平台統計**
- 設備熱度追蹤
- 用戶行為分析
- 轉換率追蹤
