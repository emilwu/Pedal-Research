2. ESP Throbber-CTM 
3. Greco Spacey Sounds TE-500 Thinline (改裝 Lindy Fralin Wide Range 拾音器)

## 個別吉他深度分析

### ESP EC-CTM Eclipse Custom

**核心規格 (Core Specs):**
* **琴身 (Body):** Honduras Mahogany 單切琴身 + Hard Maple Top，Multi-ply Binding (CTM 標誌)，光澤或緞面烤漆
* **琴頸 (Neck):** Honduras Mahogany，Set-Neck 結構，Thin U 輪廓 (.800"-.880")，24.75" 弦長
* **指板 (Fingerboard):** Ebony，305R 弧度 (12")，22 Extra Jumbo 不鏽鋼格子 (JESCAR FW57110-NS)，ESP Flag 珍珠貝鑲嵌
* **拾音器 (Pickups):** EMG JH HET SET - JH-N (Neck) + JH-B (Bridge) 主動式雙線圈
* **琴橋系統 (Bridge):** TonePros T3BT Locking TOM Bridge + GOTOH 510FA Tailpiece
* **特殊功能 (Special Features):** Carbon Nut (42mm)，GOTOH SG301-04 MG-T 鎖定旋鈕，3-Way 撥鈕，2 Volume + 1 Tone 控制，Deluxe Hard Case

**音色與演奏特性 (Tonal & Playing Characteristics):**
✅ **EMG JH HET SET 革命性音色：** JH-B 採用 Steel Pole Pieces (非傳統 Bar Magnets)，提供比 EMG 81 更緊實的 Attack，更豐富的低頻與延展的高頻響應
✅ **優異的動態範圍：** JH SET 結合了被動拾音器的清晰度與主動拾音器的高輸出，比傳統 81/60 SET 有更好的 Volume Knob 反應與和弦分離度
✅ **完美的拾音器平衡：** JH-N 採用 Ceramic Pole Pieces 與更大的 Core，比 EMG 60 有更多 Attack、更高輸出與更豐滿的低頻
✅ **Clean 音色優異：** 相較 EMG 81 在 Clean 下的限制，JH SET 提供更溫暖、更有音樂性的 Clean Tone，Chime 般的清晰度
✅ **Eclipse CTM 極致工藝：** 日本 Custom Shop 手工製作，Plek 處理，Multi-ply Binding，Ebony 指板，代表 ESP 最高品質標準
❌ **主動拾音器電池需求：** 需要 9V 電池供電，演出前需確認電量
❌ **高輸出特性：** 雖然比 81/60 更適合 Clean，但仍偏向 Modern Rock/Metal 方向，Vintage Blues 風格演奏者可能覺得過於強勁

**輸出規格 (Output Specs):**
* **JH-B 橋拾音器：** Steel Pole Pieces，輸出約 13.5k (比 EMG 81 更豐滿)
* **JH-N 頸拾音器：** Ceramic Pole Pieces，輸出約 10.5k (比 EMG 60 更有 Attack)
* **阻抗：** 25k (主動拾音器標準)，需使用 EMG 專用 25k 音量與音調旋鈕

**可切換設定 (Switching Options):**
* **3-Way Toggle Switch:**
  - Position 1: 橋拾音器 (JH-B) - 緊實、強力、現代金屬音色
  - Position 2: 橋+頸拾音器 (JH-B + JH-N) - 平衡、豐滿、Rhythm 音色
  - Position 3: 頸拾音器 (JH-N) - 溫暖、厚實、Lead/Jazz 音色
* **2 x Volume Controls:** 獨立控制每個拾音器輸出
* **1 x Tone Control:** Master Tone，微調高頻響應

**音色描述:**

*Clean 音色：* JH SET 在 Clean 設定下展現出主動拾音器少見的音樂性與溫暖度。頸拾音器提供 Chime 般的清晰度與豐滿的低頻，非常適合 Jazz Chord 與 Neo-Soul 演奏。橋拾音器保持清晰的 Attack 與良好的弦分離度，適合 Funk Rhythm 與清晰的 Arpeggio。相較 EMG 81/60 組合，JH SET 的 Clean 音色更自然、更溫暖，Volume Knob 的動態反應更靈敏。

*Crunch/Overdrive：* 這是 JH SET 最閃耀的領域。橋拾音器提供緊實、聚焦、穿透力強的 Crunch 音色，非常適合現代搖滾的 Palm Mute Riff 與 Power Chord。相較 EMG 81 的 Searing、Screaming 80 年代音色，JH SET 提供更大、更肥、更有 Growl 的 90 年代至今的現代音色。頸拾音器在 Overdrive 下保持厚實的低中頻與滑順的高頻，非常適合 Lead 演奏。

*High-Gain：* JH SET 專為高增益演奏設計。比 EMG 81/85 有更多的低頻、低中頻與延展的高頻，但保持緊實、聚焦的特性，不會變得模糊或混濁。非常適合現代金屬、Djent、Progressive Metal 等需要極致清晰度與 Punch 的風格。六弦開放和弦的弦分離度優異，複雜的 Chord Voicing 也能保持清晰。

**選擇理由/考量:**
ESP EC-CTM 搭配 EMG JH HET SET 是現代高增益演奏的終極選擇。日本 Custom Shop 的極致工藝，搭配 James Hetfield 與 EMG 多年合作開發的革命性拾音器，提供了傳統主動拾音器無法達到的動態範圍與音色多樣性。Honduras Mahogany + Maple Top 的經典木材組合提供溫暖的基底與明亮的 Attack，Ebony 指板增添清晰度與快速的觸弦反應。24.75" 弦長提供溫暖、厚實的音色與舒適的推弦手感。這是一把從 Clean Jazz 到 Extreme Metal 都能勝任的全能型吉他，特別適合需要高品質、高可靠性、高輸出的專業演奏者。

### ESP Throbber-CTM

**核心規格 (Core Specs):**
* **琴身 (Body):** Semi-Hollow 半空心設計，Honduras Mahogany Back，特殊 Sound Reservoir 空腔設計
* **琴頸 (Neck):** Mahogany，Set-Neck 或 Set-Thru 結構 (視年份)，Slim U 或 Thin U 輪廓
* **指板 (Fingerboard):** Ebony (CTM 版本)，305R 弧度，22 frets，ESP Flag 或特殊鑲嵌
* **拾音器 (Pickups):** (Neck) Seymour Duncan APH-1n，(Bridge) Seymour Duncan TB-APH-1b
* **琴橋系統 (Bridge):** TonePros Locking TOM Bridge + Tailpiece
* **特殊功能 (Special Features):** Sound Reservoir 空腔設計 (護板下方與琴橋下方)，高音弦側較大空腔、低音弦側較小空腔以保持緊實度，橋拾音器低音側延伸設計，頸拾音器高音側加大設計

**音色與演奏特性 (Tonal & Playing Characteristics):**
✅ **Sound Reservoir 創新設計：** 護板下方與琴橋下方的特殊空腔設計，提供半空心吉他的共鳴特性，但保持 Solid Body 的緊實度與 Feedback 控制
✅ **APH-1 Alnico II Pro 經典音色：** 溫暖、甜美、中等輸出的 PAF 風格雙線圈，非常適合 Jazz、Blues、Classic Rock 與 Slide Guitar
✅ **優異的自然延音：** Alnico II 磁鐵與中等纏繞提供更自然的弦振動，延音優美而不過度壓縮
✅ **Clean 音色專家：** APH-1 設計理念是從音箱獲得失真而非拾音器，因此 Clean 音色非常純淨、溫暖，Attack 較軟
✅ **半空心共鳴與清晰度：** 即使在高增益失真下，半空心結構仍能提供清晰、分離度良好的音色
❌ **中等輸出限制：** APH-1 的中等輸出可能不適合 High-Gain Metal 或極度失真的音樂風格
❌ **半空心 Feedback 風險：** 雖然 Sound Reservoir 設計有助控制，但在極高音量下仍可能產生 Feedback

**輸出規格 (Output Specs):**
* **APH-1n 頸拾音器：** DC Resistance 約 7.75k，Alnico II 磁鐵，溫暖、滑順、充滿自然延音
* **TB-APH-1b 橋拾音器：** DC Resistance 約 8.0k (Trembucker Spacing)，Alnico II 磁鐵，平衡的輸出，優異的清晰度與 Articulation
* **阻抗：** 500k (被動拾音器標準)

**可切換設定 (Switching Options):**
* **3-Way Toggle Switch:**
  - Position 1: 橋拾音器 (TB-APH-1b) - 明亮、清晰、Classic Rock Lead
  - Position 2: 橋+頸拾音器 (TB-APH-1b + APH-1n) - 平衡、豐滿、Rhythm/Crunch
  - Position 3: 頸拾音器 (APH-1n) - 溫暖、滑順、Jazz/Blues Lead
* **Volume Controls:** 可能為 2 Volume (獨立) 或 Master Volume (視具體配置)
* **Tone Controls:** 可能為 2 Tone 或 Master Tone

**音色描述:**

*Clean 音色：* APH-1 SET 在 Clean 設定下展現其最佳特質。頸拾音器提供溫暖、滑順、充滿泛音的 Jazz Tone，非常適合 Chord Melody 與 Ballad Solo。橋拾音器保持清晰的 Attack 但不刺耳，非常適合 Country、Blues、Classic Rock 的 Rhythm 演奏。半空心結構增添木質共鳴與空氣感，使 Clean 音色更立體、更有深度。

*Crunch/Overdrive：* 這是 Throbber-CTM 最擅長的領域。APH-1 設計理念是從音箱獲得失真，因此在中度失真下展現極佳的音樂性。頸拾音器提供厚實、溫暖的 Blues Lead Tone，非常適合推弦與滑奏。橋拾音器提供清晰、有 Bite 的 Classic Rock Tone，Articulation 優異，和弦分離度極佳。半空心結構增添的共鳴使音色更生動、更有表情。

*High-Gain：* APH-1 並非為極度失真設計，但仍能處理中高增益的應用。音色偏向溫暖、厚實，而非現代金屬的緊實、Aggressive 特性。半空心結構在高增益下保持良好的清晰度，不會變得模糊，但可能在極高音量下產生 Feedback。最適合 Classic Rock、Hard Rock、Blues Rock 等偏向 Vintage 的失真音色。

**選擇理由/考量:**
ESP Throbber-CTM 是一把獨特的半空心設計吉他，結合了 Solid Body 的緊實度與 Hollow Body 的共鳴特性。Sound Reservoir 的創新設計提供了最佳的平衡點 - 足夠的共鳴增添音色深度，但不會犧牲 Feedback 控制。Seymour Duncan APH-1 SET 是經典的 PAF 風格拾音器，非常適合 Jazz、Blues、Classic Rock 等需要溫暖、自然音色的風格。這把琴在 Emil 的收藏中扮演著重要的互補角色 - 當 EC-CTM 專精現代高增益，Throbber-CTM 則專精 Vintage、溫暖、動態豐富的音色。非常適合錄音室應用，特別是需要木質共鳴與空氣感的演奏。

### Greco Spacey Sounds TE-500 Thinline (改裝 Lindy Fralin Wide Range Set)

**核心規格 (Core Specs):**
* **琴身 (Body):** Semi-Hollow Ash (日本 Sen Ash)，Thinline 設計，Natural 或 Sunburst 光澤硝基漆面
* **琴頸 (Neck):** One-Piece Maple，Bolt-On 結構，C 型或 Slim 輪廓，42mm 弦枕寬度
* **指板 (Fingerboard):** Maple，7.25" 弧度，21 Vintage Frets，25.5" 弦長 (628mm)
* **拾音器 (Pickups):** Lindy Fralin Wide Range Humbucker Set (改裝，原廠為 Maxon Wide Range Humbuckers)
* **琴橋系統 (Bridge):** Vintage-Style Telecaster 3-Saddle Bridge，String-Through-Body
* **特殊功能 (Special Features):** 1970 年代 Fujigen-Gakki 工廠製造，"Lawsuit Era" 傳奇品質，F-Hole 設計，Greco 原廠鍍鉻硬體

**音色與演奏特性 (Tonal & Playing Characteristics):**
✅ **Lindy Fralin Wide Range 現代演繹：** 清晰、明亮、Articulate 的音色，比原廠 CuNiFe Wide Range 更平衡、更現代，保持 Wide Range 特有的 Fender Twang 與 Humbucker 豐滿度
✅ **大、彈跳、清晰的低頻：** Fralin Wide Range 的低頻特性類似 Fender Humbucker 設計，大而彈跳但清晰，不會變得模糊
✅ **Slightly Scooped Mids with High-Mid Emphasis：** 中頻稍微 Scooped，但高中頻強調，確保 Cut Through the Mix 而不會尖銳、刺耳或單薄
✅ **Clean 音色：loud、bright、articulate；Dirty 音色：smooth、slightly crunchy**
✅ **Fujigen "Better than Fender" 傳奇工藝：** 1970 年代日本 Fujigen 工廠的製造品質常被認為超越同期 CBS Fender，這把琴代表日本 "黃金時代" 的極致工藝
✅ **半空心 Ash 共鳴：** 日本 Sen Ash 琴身提供明亮、清晰的音色基底，半空心結構增添共鳴與空氣感
❌ **復古 7.25" 弧度：** 對現代快速演奏與推弦技巧可能有限制
❌ **Bolt-On 42mm 窄弦枕：** 對手大的演奏者可能略窄

**輸出規格 (Output Specs):**
* **Fralin Wide Range 頸拾音器：** DC Resistance 約 7.5-8.0k，AlNiCo V 磁鐵，清晰、明亮、平衡
* **Fralin Wide Range 橋拾音器：** DC Resistance 約 8.5-9.0k，AlNiCo V 磁鐵，更多 Midrange Bite，適合 Cut Through
* **阻抗：** 500k (被動拾音器標準)
* **音色特性：** 類似 Fralin P-92 - Bold、Clear、Articulate

**可切換設定 (Switching Options):**
* **3-Way Toggle Switch:**
  - Position 1: 橋拾音器 (Fralin Wide Range Bridge) - Bright、Biting、Classic Tele Lead
  - Position 2: 橋+頸拾音器 (Series) - Thick、Full、Unique Wide Range Mix
  - Position 3: 頸拾音器 (Fralin Wide Range Neck) - Warm、Fat、Bassy Lead/Jazz
* **2 x Volume Controls:** 獨立控制每個拾音器
* **2 x Tone Controls:** 獨立音調控制

**音色描述:**

*Clean 音色：* Lindy Fralin Wide Range 在 Clean 設定下展現其最佳特質。頸拾音器提供 Fat、Bassy 的 Jazz Tone，比傳統 Single-Coil 更豐滿但保持清晰度。橋拾音器提供 Full of Midrange Bite 的 Classic Tele Lead Tone，非常適合 Country、Funk、Pop 的清晰 Attack。中間檔位（兩個拾音器同時）提供獨特的 Wide Range Mix Tone，既有 Humbucker 的豐滿度又有 Fender 的 Twang 特性。半空心 Ash 琴身增添木質共鳴與明亮度，使 Clean 音色非常適合錄音。

*Crunch/Overdrive：* Fralin Wide Range 在中度失真下展現極佳的平衡性。相較傳統 Humbucker，Wide Range 不會過早 Break Up，保持厚實的低中頻與滑順的高頻。橋拾音器提供 Beefy 但 Clear 的 Hard Rock Tone，和弦分離度優異。頸拾音器提供溫暖、Smooth 的 Blues Lead Tone，Articulation 極佳。中間檔位在 Crunch 下特別有趣，提供既有 Twang 又有 Crunch 的獨特音色，非常適合 Classic Rock Rhythm。

*High-Gain：* Fralin Wide Range 雖非為極度失真設計，但仍能處理 Hard Rock 的高增益應用。Slightly Scooped Mids 使其在 Band Mix 中容易突出，High-Mid Emphasis 確保 Lead 清晰可聞。半空心結構在高增益下保持良好控制，不會產生過度 Feedback。最適合 70 年代 Hard Rock、Classic Rock、Blues Rock 等需要 Vintage Humbucker Crunch 的風格。

**選擇理由/考量:**
Greco Spacey Sounds TE-500 改裝 Lindy Fralin Wide Range Set 是一把極具歷史意義與音色獨特性的吉他。1970 年代 Fujigen 工廠的製造品質代表日本 "Lawsuit Era" 的黃金時代，常被認為超越同期 CBS Fender。改裝 Lindy Fralin Wide Range 拾音器使其音色更現代、更平衡，同時保持 Wide Range 特有的 Fender Twang 與 Humbucker 豐滿度。這是一把在 Emil 收藏中扮演重要角色的吉他 - 提供 EC-CTM 與 Throbber-CTM 都無法達到的 Vintage Tele Twang 與 Wide Range 獨特音色。非常適合 Neo-Soul、Funk、Jazz、Classic Rock 等需要清晰、Articulate、有 Character 的音色。

### Fender Made in Japan Flagship Tokyo Editions Telecaster® Thinline

**核心規格 (Core Specs):**
* **琴身 (Body):** Semi-Hollow Mahogany Back + Maple Top，Binding，Satin Polyester 烤漆，F-Hole 設計
* **琴頸 (Neck):** Maple，Soft "V" 輪廓，Satin Urethane 烤漆
* **指板 (Fingerboard):** Selected Rosewood，9.5" 弧度 (241mm)，22 Narrow Tall Frets，White Pearloid Dot 鑲嵌
* **拾音器 (Pickups):** (Neck) Seymour Duncan SP90-1N Vintage P90，(Bridge) Seymour Duncan SP90-1 Vintage P90
* **琴橋系統 (Bridge):** 3-Saddle Vintage-Style Strings-Through-Body Tele with Slanted Brass Barrel Saddles
* **特殊功能 (Special Features):** Matching Headstock，Gold Hardware，1-Ply Gold 或 1-Ply Tortoiseshell Pickguard，Bone Nut (1.615" / 41.02mm)，25.5" 弦長 (648mm)

**音色與演奏特性 (Tonal & Playing Characteristics):**
✅ **Seymour Duncan SP90-1 Vintage P90 經典音色：** 溫暖、Mellow 的音色基底，但保持明亮、華麗的 Lead Tone 穿透力
✅ **Mahogany/Maple 半空心結構：** 比傳統 Ash Thinline 更溫暖、更厚實，非常適合 Neo-Soul、Crossover、Jazz 等現代音樂風格
✅ **P90 獨特的 Midrange Character：** 比 Single-Coil 更厚實但比 Humbucker 更清晰，提供獨特的 Growl 與 Bite
✅ **Soft "V" 琴頸舒適握感：** 自然貼合手掌，高演奏性與舒適性並存
✅ **9.5" 現代弧度 + 22 Frets：** 比復古 7.25" 更適合現代演奏技巧，22 格提供更寬廣的音域
✅ **Flagship Tokyo 限定稀有性：** Fender 世界首家大型旗艦店限定，Matching Headstock + Gold Hardware 豪華外觀
❌ **P90 固有噪音：** Vintage P90 在高增益下可能產生 Single-Coil 噪音
❌ **半空心 Feedback 風險：** 極高音量下可能產生 Feedback

**輸出規格 (Output Specs):**
* **SP90-1N 頸拾音器：** DC Resistance 約 8.0-8.5k，AlNiCo V 磁鐵，Vintage P90 音色
* **SP90-1 橋拾音器：** DC Resistance 約 8.5-9.0k，AlNiCo V 磁鐵，稍高輸出
* **阻抗：** 500k (被動拾音器標準)

**可切換設定 (Switching Options):**
* **3-Position Blade Switch:**
  - Position 1: 橋拾音器 (SP90-1) - Bright、Cutting、Lead Tone
  - Position 2: 橋+頸拾音器 - Thick、Balanced、Rhythm
  - Position 3: 頸拾音器 (SP90-1N) - Warm、Thick、Mellow Lead
* **2 x Volume Controls (推測):** 獨立控制每個拾音器
* **2 x Tone Controls (推測):** 獨立音調控制

**音色描述:**

*Clean 音色：* Seymour Duncan SP90-1 Vintage P90 在 Clean 設定下展現其經典的 P90 特質。頸拾音器提供溫暖、Mellow、厚實的 Jazz Tone，比 Single-Coil 更豐滿但保持清晰度，非常適合 Neo-Soul 的 Chord Voicing。橋拾音器提供明亮、華麗、有 Bite 的 Lead Tone，穿透力強但不刺耳。Mahogany/Maple 半空心結構增添溫暖的共鳴與木質感，使 Clean 音色非常適合現代流行音樂、Neo-Soul、Crossover 等需要溫暖但清晰的音色。

*Crunch/Overdrive：* P90 在中度失真下展現其最佳特質。橋拾音器提供獨特的 Growl 與 Bite，比 Humbucker 更 Open、更 Dynamic，比 Single-Coil 更厚實、更有 Punch。頸拾音器提供溫暖、Smooth 的 Blues Lead Tone，Articulation 極佳，非常適合推弦與滑奏。中間檔位提供平衡的 Rhythm Tone，既有厚度又有清晰度，非常適合 Classic Rock、Blues Rock、Garage Rock 等風格。

*High-Gain：* Vintage P90 在高增益下會展現其 Single-Coil 的特性 - 可能產生噪音，音色可能略顯 Thin。但如果控制得當，P90 的獨特 Midrange Character 在高增益下仍能提供有 Character 的 Lead Tone，比 Humbucker 更 Raw、更 Aggressive。最適合 Punk、Garage Rock、Alternative Rock 等需要 Raw、Aggressive 音色的風格。

**選擇理由/考量:**
Fender Made in Japan Flagship Tokyo Editions Telecaster® Thinline 是一把極具稀有性與現代設計的吉他。Fender 世界首家大型旗艦店限定，Matching Headstock + Gold Hardware 的豪華外觀，加上 Mahogany/Maple 半空心結構與 Seymour Duncan P90 拾音器，使其成為一把既有 Vintage Vibe 又有現代功能的獨特樂器。在 Emil 的收藏中，這把琴提供了其他三把都無法達到的 P90 獨特音色 - 比 Single-Coil 更厚實，比 Humbucker 更清晰，比 Wide Range 更有 Growl。非常適合 Neo-Soul、Jazz、Fusion、Alternative Rock 等需要溫暖但有 Bite 的音色。

## 總結與建議

Emil 的四把吉他收藏展現了極高的音色互補性與風格覆蓋度：

### 收藏核心優勢

1. **完整的輸出範圍：** 從 Throbber-CTM 的自然中等輸出到 EC-CTM 的主動高輸出
2. **拾音器類型完整：** 主動 Humbucker、被動 PAF Humbucker、Wide Range Humbucker、P90 - 四種截然不同的 Character
3. **琴身結構多樣：** Solid Body + 三種不同的 Semi-Hollow 設計
4. **木材組合豐富：** Mahogany + Maple、Mahogany Semi-Hollow、Ash Semi-Hollow、Mahogany + Maple Semi-Hollow
5. **風格覆蓋全面：** 從 Jazz 到 Extreme Metal，從 Neo-Soul 到 Punk，幾乎所有風格都能勝任

### 最佳二琴組合建議

**情境一：全能型演奏者**
- **組合：ESP EC-CTM + Greco TE-500**
- **理由：** Les Paul + Tele 經典組合，現代 + Vintage 完美對比，覆蓋最廣音樂風格

**情境二：Neo-Soul/Jazz 專精**
- **組合：Greco TE-500 + Fender Tokyo Thinline**
- **理由：** Wide Range + P90 兩種不同的 "準 Humbucker" Character，都極適合 Neo-Soul

**情境三：Modern Rock 專業演奏者**
- **組合：ESP EC-CTM + ESP Throbber-CTM**
- **理由：** 高增益 + 溫暖 Clean，同品牌一致性，覆蓋從 Clean 到 Extreme 的所有需求

### 購買優先順序建議 (如需擴充收藏)

Emil 目前的四把吉他已經非常完整，但如果要進一步擴充：

1. **Stratocaster 類型吉他** - 目前缺少純正 Single-Coil Strat 音色
   - 建議：PRS Silver Sky Rosewood 或 Suhr Classic S HSS
   - 理由：補足 Single-Coil 的 Quack、清澈、動態響應

2. **現代 Superstrat** - 如需技術性演奏
   - 建議：Suhr Modern Satin 或 ESP Horizon
   - 理由：24 格、Floyd Rose、HSH 配置

3. **Acoustic-Electric** - 如需 Acoustic 音色
   - 建議：Taylor 或 Martin 電木吉他
   - 理由：錄音與現場演出的 Acoustic 需求

### 維護與保養建議

1. **ESP EC-CTM：** 定期檢查 EMG 電池電量，保持琴橋鎖定螺絲緊固
2. **ESP Throbber-CTM：** 半空心結構需注意濕度控制，避免木材變形
3. **Greco TE-500：** 1970 年代 Vintage 吉他，硝基漆面需小心保養，避免溫濕度劇變
4. **Fender Tokyo Thinline：** 定期清潔 Gold Hardware，避免氧化

Emil 的吉他收藏不僅展現了深厚的音樂知識與品味，更重要的是每把琴都有其獨特的定位與不可替代性。從現代金屬到 Vintage Jazz，從 Neo-Soul 到 Classic Rock，這四把吉他能夠滿足幾乎所有音樂創作與演出需求。
